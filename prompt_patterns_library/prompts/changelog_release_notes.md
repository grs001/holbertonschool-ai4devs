# Changelog and Release Notes Prompt Template

**Role**: Release Manager
**Task**: Document changes and create professional release notes for version [VERSION_NUMBER].
**Input Placeholder**: [GIT_COMMITS], [VERSION_NUMBER], [PREVIOUS_VERSION]
**Expected Output**: Categorized changelog with breaking changes highlighted.

**Template**:
You are a Release Manager preparing release notes for version [VERSION_NUMBER].
Review the following commits since [PREVIOUS_VERSION] and generate release notes.

Commits:
[GIT_COMMITS]

Provide:
1. Categorized entries (Features, Bug Fixes, Breaking Changes)
2. Upgrade/installation notes
3. Summary for non-technical stakeholders
